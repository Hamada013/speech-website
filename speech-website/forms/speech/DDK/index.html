Instructions

Time the number of seconds it takes the client to complete each task the prescribed number of repetitions using the built-in stopwatch. Normative values are provided for both pediatric clients (under 7 years) and older children (6-13 years). For pediatric clients, norms are expressed as repetitions per second (rps) based on Hickman (1997) and Robbins & Klee (1987), while for older children, norms are given as the number of seconds to complete a fixed set of repetitions based on Fletcher (1972).
